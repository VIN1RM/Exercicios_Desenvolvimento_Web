Entendendo os Títulos

O que são Títulos?

Os títulos são usados para organizar o conteúdo de uma página, indicando a hierarquia das informações.

Tipos de Títulos

Existem seis níveis de títulos, de <h1> a <h6>, sendo <h1> o mais importante.

Exemplo de Uso

Use <h1> para o título principal, <h2> para seções, <h3> para subseções e assim por diante.
